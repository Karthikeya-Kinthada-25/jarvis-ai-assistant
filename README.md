# Jarvis AI Assistant

A voice-controlled desktop automation system that executes system-level tasks using natural language.

## Features
- Open applications (Chrome, Notepad, Calculator)
- Web automation (search, navigation)
- Context-based command handling
- Voice interaction system

## Example Commands
- Open Chrome
- Search Python tutorials
- Open second result
- Play first video

## Tech Stack
- Python
- Speech Recognition
- OS Automation
